## Style Guidelines:

- Primary color: Soft blue (#64B5F6) for a calm and trustworthy feel.
- Background color: Very light blue (#E3F2FD), almost white.
- Accent color: Muted purple (#9575CD) to complement the blue and add sophistication.
- Body and headline font: 'PT Sans' (sans-serif) for a clean and modern look.
- Clean and intuitive layout with clear separation of income and expense sections.
- Use simple, easily recognizable icons for expense categories (e.g., food, transport, entertainment).
- Subtle transitions and animations when adding or viewing expenses for a smooth user experience.
